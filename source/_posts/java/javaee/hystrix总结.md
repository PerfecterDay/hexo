hystrix 编程的一般流程
1. 构造一个调用依赖服务的 command
   1. 继承 HystrixCommand 类，重写 `R run()` 方法实现调用逻辑
   2. 继承 HystrixObservableCommand  类，重写 `Observable<String> construct()` 方法

2. 为 command 配置 fallback
   重写 `R getFallback()` 方法即可

3. 调用 command 的执行方法即可
